RSV – AI-Powered Restaurant Reservation Bot

A production-ready restaurant reservation chatbot with AI-powered conversation, smart table management, and a responsive web UI.

Overview

RSV is a full-stack AI chatbot that handles restaurant reservations like a human assistant. It understands natural language, manages table availability, prevents double bookings, and provides a smooth booking experience.

Features

Smart Chatbot

  • ML-based intent classification (TF-IDF + Naive Bayes)
  • 8 intents: greeting, menu, booking, availability, cancel, modify, FAQ, feedback
  • Confidence-based responses with fallback handling

Entity Extraction

  • Date parsing: today, tomorrow, weekdays, explicit dates
  • Time parsing: 12-hour and 24-hour formats
  • Guest count extraction
  • Customer name detection

Availability Engine

  • Real-time table availability
  • Double booking prevention
  • Time-slot-based reservations
  • Alternative slot suggestions
  • Table capacities: 2, 4, 6, 8

Reservation System

  • Create, modify, cancel reservations
  • Unique confirmation IDs
  • Pre-order dishes
  • Special requests handling

Menu System

  • 56 items across 5 categories
  • Vegan / vegetarian flags
  • Popularity ratings
  • Chef’s specials
  • Price filtering and search

AI / ML

  • Intent classification using scikit-learn
  • Sentiment analysis using TextBlob
  • Popularity-based recommendations
  • Coupon code support

UI

  • Mobile-first responsive design
  • Dark / light mode
  • Real-time chat with typing indicators
  • Menu cards with images
  • Reservation modal and testimonials

Tech Stack

  • Backend: Python, FastAPI
  • Frontend: HTML, CSS, JavaScript
  • AI / ML: scikit-learn, TextBlob, NLTK
  • Database: SQLite (cloud-ready)

API Endpoints

  • POST /chat
  • GET /menu
  • POST /menu/search
  • GET /menu/chef-specials
  • GET /menu/popular
  • POST /availability
  • POST /reservation
  • PUT /reservation/{id}
  • DELETE /reservation/{id}
  • POST /feedback
  • GET /admin/stats
  • GET /health

Installation

cd rsv-restaurant-bot
pip install -r requirements.txt
python -m textblob.download_corpora
python database/seed.py
python -c "from chatbot.intent_model import IntentModel; m = IntentModel(); m.train()"
python app.py

Application runs at: http://localhost:8000

Testing

pytest tests/test_availability.py -v
